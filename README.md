# Medyanes Full Stack Developer Task - Todo App

Merhaba, bu proje Medyanes tarafından verilen "Full Stack Developer" task'ı kapsamında geliştirdiğim CRUD uygulamasıdır.

Task sürecinde sadece frontend değil backend tarafında da Next.js API Routes aktif rol alarak Full Stack bir yapı kurmaya çalıştım.

Projeye başlamadan önce paylaşılan kaynakları detaylıca inceledim:

- Medyanes 360 Backend Dokümanı'nı okudum.
- Next.js Backend CRUD videosunu izleyip mantığı kavradım.
- CaliskanAri örnek reposundaki dosya yapısını özellikle services ve api klasörlerini analiz ettim.

Ayrıca AI araçlarını da kullanarak kendime bir yol haritası (roadmap) çizdim ve adım adım ilerledim.

- pages/api/todos.ts üzerinden GET, POST, PUT, DELETE metodlarını yazdım.
- Prisma ile Todo modelini oluşturdum ve MongoDB Atlas'a bağladım.
- Servis katmanını fetchApi kullanarak backend ile frontend'i konuşturdum.
- Zustand ile yapılan değişikliklerin ekleme, silme vs. anlık olarak arayüze yansımasını sağladım.
- Mobilde ve masaüstünde düzgün çalışan, göz yormayan modern bir arayüz tasarladım. Daha fazla üstüne düşebilirdim ui kısmında ama task'daki amacın dışına çıkardım.
